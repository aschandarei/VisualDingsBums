# VisualDingsBums

###Brain F\*ck IDE

#### Features
- syntax coloring
- step-by-step execution
- debug
- visual programming using the Brain Vixx tool
- Use your own custom BF dialect
- Convert source code between BF dialects

Created in .Net 4.8, C# and WPF
